OBTENIR VOTRE CLÉ API (5 MIN)
1
Aller sur openrouter.ai
Ouvrez votre navigateur et tapez openrouter.ai dans la barre d'adresse.
2
Cliquer "Sign Up"
Inscrivez-vous avec votre email ou via Google. Aucune carte bancaire nécessaire.
3
Aller dans "Keys"
Dans le menu de navigation, cliquez sur Keys puis Create Key. Nommez-la n8n-automation.
4
Copier la clé immédiatement !
Elle commence par sk-or-v1-... Elle ne s'affiche qu'une seule fois. Copiez-la dans un fichier texte.
5
Configurer dans n8n
Settings → Credentials → Add Credential → Header Auth → Nom: OpenRouter-Free · Header: Authorization · Valeur: Bearer + votre clé
